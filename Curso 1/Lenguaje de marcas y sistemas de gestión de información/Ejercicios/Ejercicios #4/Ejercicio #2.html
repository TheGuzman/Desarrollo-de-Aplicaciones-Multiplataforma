Introducción

SQL, Strcuture Query Language (Lenguaje de consulta Estructurado) es el lenguaje de programación para trabajar con bases de datos relacionales como MySQL, Oracle, etc. MySQL es un interpretador de SQL, es un servidor de bases de datos.
