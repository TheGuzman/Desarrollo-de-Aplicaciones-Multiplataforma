Declarar las variables adecuadas para los tipos anteriores con identificadores
válidos e inicializarlas con esos valores.

a) 54 -> entero:
    int a = 54;

b) 45776 -> entero:
    int b = 45776;

c) ‘X’ -> carácter:
    char c = 'X';

d) true -> buleano:
    boolean d = true;

e) 557678.568 -> decimal:
    float e = 557678.568;
