Ordenamiento y Búsqueda

Descripción:

Debes crear un programa en C++ que haga uso de los algoritmos proporcionados por la 
biblioteca <algorithm> para ordenar y buscar elementos en una lista de números enteros.

Requisitos:

1. El programa debe contener una función para generar una lista de números enteros 
aleatorios. El usuario debe especificar la cantidad de números y el rango (mínimo y máximo).

2. Debes utilizar los algoritmos de <algorithm> para realizar lo siguiente:
   - Ordenar la lista de números de manera ascendente.
   - Ordenar la lista de números de manera descendente.
   - Buscar un número específico en la lista y mostrar su posición si se encuentra.

3. El programa debe mostrar la lista de números antes y después de cada operación de 
ordenamiento y mostrar el resultado de la búsqueda.

4. Utiliza un menú para permitir al usuario elegir entre las diferentes operaciones y 
salir del programa.

4. Buscar Número
5. Salir
