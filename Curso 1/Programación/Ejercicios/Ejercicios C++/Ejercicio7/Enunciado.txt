Título de la Tarea: Estructura de Datos Genérica - Pila

Descripción:

Se te ha asignado crear una estructura de datos genérica tipo pila utilizando C++ con plantillas (templates). La pila deberá ser 
capaz de almacenar elementos de cualquier tipo de datos. Debes implementar las siguientes funcionalidades para la pila genérica:

1. Agregar un elemento a la pila.
2. Sacar un elemento de la pila.
3. Verificar si la pila está vacía.
4. Mostrar el elemento en la cima de la pila sin eliminarlo.
5. Mostrar todos los elementos de la pila.
6. Salir del programa.

Requisitos:

1. Implementa la pila utilizando plantillas de C++ para que pueda almacenar elementos de cualquier tipo de datos (por ejemplo,
 enteros, números de punto flotante, cadenas de caracteres, objetos personalizados).

2. Utiliza una estructura de datos dinámica (por ejemplo, un arreglo de memoria dinámica o una lista enlazada) para almacenar 
los elementos en la pila.

3. Implementa una interfaz basada en menús que permita al usuario elegir entre las operaciones mencionadas anteriormente.

4. Utiliza un manejo adecuado de errores para tratar casos como sacar elementos de una pila vacía.
