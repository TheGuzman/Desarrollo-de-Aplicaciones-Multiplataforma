Se adjunta un script SQL de la creación de la base de datos 'CINE' con dos tablas: PELICULAS Y SALAS. Este script se corresponde al fichero: script_bd_peliculas.sql

Deberás ejecutar este script para disponer de la base de datos en MySQL y poder solucionar las consultas que se proponen en el fichero: examen bd peliculas_enunciado.sql

Se deberá entregar el fichero examen bd peliculas_enunciado.sql, con la solución de cada consulta justo debajo de su enunciado. Renombrar este fichero con la solución con el nombre Apellidos_Nombre_solucion.sql
