Ejercicio 1:

Bases de datos jerárquicas:

    Es un modelo de bases de datos en que estos se organizan en forma de árbol invertido, en donde un nodo de información llamado padre puede tener varios hijos, uno o ninguno. Como ejemplo tenemos:

    - Adabas.
    - IMS.
    - Focus.

Bases de datos relacionales:

    Es un modelo de bases de datos basado en las 12 reglas de Codd. Muy 
utilizado actualmente para la representación de problemas reales y 
administración de datos de forma dinámica. Como ejemplo tenemos:

    - MySQL.
    - Oracle.
    - PostgresSQL.

Bases de datos orientadas a objetos:

    Es un modelo de bases de datos bastante reciente basado en los modelos
informáticos orientados a objetos (paradigma de programación POO). Este 
modelo trata de almacenar los objetos completos en la base de datos 
("estado" y "comportamiento"). Como ejemplo tenemos:

    - ObjectBD
    - Zope Object Database
    - Persistencia
